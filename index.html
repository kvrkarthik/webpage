Process to Install WSL on Windows

Step 1:

Open Command Prompt on your machine.

Step 2:

Type wsl --install and press Enter.

Step 3:

Give all the necessary permissions to download. Let it download Ubuntu even.

Step 4:

Wait for the installation to complete. After installation, please restart.

Step 5:

After restarting, Command prompt will be opened automatically and launches Ubuntu.

Step 6:

Here, You create your own Unix user by giving username and password.
Note: Please remember the username and password as you have to use it further.

Step 8:

To launch Ubuntu, open Command Prompt and enter ubuntu.exe and press Enter.

Step 9:

Now you can use Unix/Linux Commands on it.

Install other Distributions

To install other ones, type wsl --install -d <Distribution Name>. Replace <Distribution Name> with the name of the distribution you want to install such as Kali-Linux, Debian, etc.

For other info, Please refer this website by Microsoft.
